A palindrome is a word or phrase that says the same thing forwards and backwards. This was a really fun exercise while learning JavaScript! The code is structured in three functions.

getMessage

Gets the word or phrase one wishes to check. There is a validation checker in there to make sure no one is passing and empty string to the program! Once this is checked, I used a REGEX to convert all data to lower case string.

Reverse Message

Once the word or phrase has been REGEX'd, the reverseMessage function reverses the string. The two values are then compared.

Display Message

Finally, we display the message to the user, informing them if their word or phrase is a palindrome or not.
